정규식(Regular Expression) 연습

문자열 : 

    형식 : /~~/ 플러그인 , /[~~]/플러그인
    자바 : ^[~~]$

    * 플러그인 
      g(global) : 전체범위에서의 거리
      m(multi line) : 여러줄에 걸쳐서 처리(textarea 태그)
      s(single line) : 한 줄에서의 처리(기본값으로 샹락)
      i : 영문자인 경우 대/소문자의 구별을 하지 않는다.
      u(unicode) : 유니코드에 대한 구별 처리할 때 사용 (사용 안 함) 

    * 정규식을 실해하기 위해서 사용되는 함수 
      1. 사용자 입력 문자열 : .match(정규 표현식 or 정규 표현식의 변수) // 포함 유무에 대한 비교시에 사용하면 편리
      2. 정규 표현식 변수 : .test(사용자 입력 문자열) // 구체적인 문자(열)간의 비교 체크에 사용하면 편리

Front에서 서버로 값을 전송(Form태그 사용)할시는 사용자 값을 입력 후 반드시 유효성 검사를 거쳐야 한다.
